# Time-Calculation-for-Sorting-Algorithms

A **Python 3** code for time calculating some Sorting Algorithm.

# Features

    1. Brick_Sort
    2. Bubble_Sort
    3. Bucket_Sort
    4. Comb_Sort
    5. Counting_Sort
    6. Cycle_Sort
    7. Gnome_Sort
    8. Heap_Sort
    9. Insertion_Sort
    10. Intro_Sort
    11. Merge_Sort
    12. Pigeonhole_Sort
    13. Quick_Sort
    14. Radix_Sort
    15. Selection_Sort
    16. Stooge_Sort
    17. Strand_Sort
    18. Tree_Sort
    
    Total 18 sorting algorithm with their execution time 
    on a Dataset with the outcome of fastest and slowest 
    algorithm on given dataset.

## Prerequisites

* **Python 3**
